Overview

This project focused on building a Python-based predictive model to assist in quarterly budget allocations. By analyzing 5 years of historical financial data, I developed an engine capable of forecasting revenue trends with over 90% accuracy, allowing for more strategic resource distribution.

Methodology

Python
NumPy
Pandas

The Challenge

The primary challenge was managing high volatility in seasonal data. Standard linear models were insufficient; therefore, I implemented a time-series decomposition approach to separate trend, seasonality, and noise before feeding the data into the final prediction algorithm.

Key Results

  • Reduced Budget Variance
  • Automated Data Cleaning
  • 5-Year Trend Analysis
